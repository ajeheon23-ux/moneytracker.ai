# Spending Pattern AI

Simple Streamlit app for daily spending tracking and pattern analysis.

## Features
- 4-category daily input (Food, Shopping, Leisure, Other)
- Monthly calendar with category colors
- Day/Week/Month spending comparisons
- Monthly/Yearly projection at current pace
- Strong feedback for overspending patterns
- Annualized spend to real-world car recommendation

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
